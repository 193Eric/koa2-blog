### 目录结构
<pre>
.
├── models                   // model层 处理数据
│	├── cagegory.js             // 处理文章数据
│   ├── comment.js              // 处理评论
│   ├── post.js                 // 处理传输数据
│	└── user.js                 // 处理用户数据
├── config                   // 数据库配置文件
├── middlewares              // 公用的中间件
├── package.json             // 项目配置文件
├── views                    // 前端目录
├── routes                   // 路由
├── .gitignore               //git忽略文件
</pre>

### 本地测试环境：  
#### 第一步：安装mongodb.可以去菜鸟教程
#### 第二步：npm install 把项目依赖安装.  
#### 第三步：npm run dev 启动项目   
#### 第四步：浏览器输入 localhost:3000
